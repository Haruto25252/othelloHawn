# othelloHawn

🎮 **プレイはこちら → https://haruto25252.github.io/othelloHawn/**

---

ブラウザだけで動く1ファイル完結のオセロゲームです。

## 機能

- **4種類の対戦モード**
  - 2人対戦（人 vs 人）
  - AI やさしい（ランダム）
  - AI ふつう（開放度理論）
  - AI むずかしい（ミニマックス深さ5）
- **レイアウト切り替え** — スマホ縦並びとPC横並びをゲーム中でも即時切り替え可能
- **配置プレビュー** — ホバー（PC）または長押し（スマホ）で裏返る石をプレビュー表示
- **ツールチップ** — 開放度・自分/相手の手数を確認できる戦略情報
- **ストラテジーヒント** — 角・X打ち・手数優位などをリアルタイム表示
- **アニメーション** — 石の配置・裏返しを波状アニメーションで演出

## 使い方

`index.html` をブラウザで開くだけで動作します。ビルド不要です。
